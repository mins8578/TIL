### 계산기 함수
```python
def calculator(a, b, operator):
    # 여기에 코드 작성
    if operator == '+':
        return a + b
    elif operator == '-':
        return a - b
    elif operator == '*':
        return a * b
    elif operator == '/':
        if b == 0:
            return("0으로 나눌 수 없습니다.")
        return a / b
    else:
        print("지원하지 않는 연산자입니다")
    pass

print(calculator(10, 5, '+'))  # 15
print(calculator(10, 5, '-'))  # 5
print(calculator(10, 5, '*'))  # 50
print(calculator(10, 5, '/'))  # 2.0
print(calculator(10, 0, '/'))  # 0으로 나눌 수 없습니다
print(calculator(10, 5, '%'))  # 지원하지 않는 연산자입니다
```
**결과**
```
15
5
50
2.0
0으로 나눌 수 없습니다.
지원하지 않는 연산자입니다
None
```

### 학생 성적 처리
```python
def print_report(name, scores):
    # 여기에 코드 작성
    print("=== 김철수 성적표 ===")
    print(f"점수: {scores}")
    avr = sum(scores) / len(scores)
    max_score = max(scores)
    min_score = min(scores)

    if avr >= 90:
        grade = "A"
    elif avr >= 80:
        grade = "B"
    elif avr >= 70:
        grade = "C"
    elif avr >= 60:
        grade = "D"
    else:
        grade = "F"
    print(f"평균: {avr:.1f}")
    print(f"최고점: {max_score}")
    print(f"최저점: {min_score}")
    print(f"등급: {grade}")
    pass

print_report("김철수", [85, 92, 78, 96, 88])

# 최고점: 96점
# 최저점: 78점
# 등급: B
```

### 비밀번호 검증
```python
# **문제**: 비밀번호가 조건을 만족하는지 검증하는 함수를 만드세요.
#
# **조건**:
#
# - 8자 이상
# - 숫자 포함
# - 대문자 포함
def validate_password(password):
    # 여기에 코드 작성
    if len(password) < 8:
        return (False, "8자 이상이어야 합니다.")

    has_digit = False
    for char in password:
        if char.isdigit():
            has_digit = True
            break
    if not has_digit:
        return (False, "숫자를 포함해야 합니다.")

    has_upper = False
    for char in password:
        if char.isupper():
            has_upper = True
            break
    if not has_upper:
        return(False, "대문자를 포함해야 합니다.")
        # 유효하면 True, 아니면 False와 이유 반환

    return (True, "유효한 비밀번호입니다.")
    pass

print(validate_password("abc"))        # (False, "8자 이상이어야 합니다")
print(validate_password("abcdefgh"))   # (False, "숫자를 포함해야 합니다")
print(validate_password("abcdefg1"))   # (False, "대문자를 포함해야 합니다")
print(validate_password("Abcdefg1"))   # (True, "유효한 비밀번호입니다")
```
**결과**
```
(False, '8자 이상이어야 합니다.')
(False, '숫자를 포함해야 합니다.')
(False, '대문자를 포함해야 합니다.')
(True, '유효한 비밀번호입니다.')
```

### 실습을 통해 알게 된 내용
> **isdigit() 함수**
> 
>문자열이 숫자로만 이루어져 있는지 여부를 확인하는 메서드입니다.
> 이 함수는 문자열 내의 하나라도 숫자 문자(0-9)로 구성되어 있을 때 True를 반환하고, 그렇지 않을 경우 False를 반환한다.
>
> **isupper() 함수**
>
> 문자열이 대문자로 이루어져 있는지 여부를 확인하는 메서드이다.
> 이 함수는 문자열 내의 하나라도 대문자로 구성되어 있을 때 True를 반환하고, 그렇지 않을 경우 False를 반환한다.
